# 🚀 Meeting Action Extractor Agent

## 🧠 Overview
This project is an AI-powered **Meeting Action Extractor Agent** built using **Microsoft Copilot Studio** and a Python backend API.

It converts meeting transcripts into structured business insights such as:
- ✅ Action items  
- ✅ Owners  
- ✅ Deadlines  
- ✅ Manager-friendly summary  

---

## ❗ Problem Statement
In most organizations, meetings generate a lot of discussion, but:
- Action items are unclear  
- Ownership is not properly assigned  
- Deadlines are missed  

This leads to poor follow-ups and reduced productivity.

---

## ✅ Solution
This agent takes meeting transcripts (from tools like Teams/Zoom or notes) and automatically extracts structured information.

👉 It helps teams:
- Track work clearly  
- Improve accountability  
- Save time after meetings  

---

## ⚙️ How It Works

1. User interacts with Copilot Agent  
2. User pastes meeting transcript  
3. Copilot sends data to backend API  
4. API processes input  
5. Structured output is returned and displayed
